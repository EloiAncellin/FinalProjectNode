Metrics app, using NodeJS, Typescript, MongoDB.

# Get Started

```
docker-compose up --build
```

# Development Environment

```
APP_ENV=dev docker-compose up --build
```

# Run tests and check coverage

```
docker exec -it web npm run converage
```
